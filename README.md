# Snake Game

A simple terminal-based snake game written in C using the ncurses library.  
Partially inspired by [wxn0brP/snake_cli](https://github.com/wxn0brP/snake_cli).

## Building and Running

### Prerequisites

- gcc
- ncurses

### Building and Running

```bash
make
```

## How to Play

`w`, `a`, `s`, `d` - move
